Blueprints

Blueprints are the building blocks of NICE. Every agent and spaceship starts as a blueprint — a template that defines its behavior, tools, and configuration.

The Catalog

NICE ships with 800+ pre-built blueprints across every industry:

  • Agent blueprints — individual AI workers (marketing analyst, content writer, support agent, etc.)
  • Spaceship blueprints — complete workflows with pre-configured crews (restaurant, law firm, agency, etc.)

Browse the full catalog at nicespaceship.com/blueprints or in the app under Bridge.

Blueprint Anatomy

Every blueprint contains:

FieldDescription
nameDisplay name
typeagent or spaceship
categoryRole category (Marketing, Engineering, Analytics, etc.)
rarityCommon, Rare, Epic, Legendary, or Mythic
descriptionWhat this blueprint does
flavorShort tagline shown on the card
configTools, model preferences, system prompt
statsPerformance metrics (speed, accuracy, capacity, power)
tagsSearchable keywords

Creating Custom Blueprints

Agent Builder

Go to Bridge > Agents > New to create a custom agent:

  1. Name your agent and pick a category
  2. Write a system prompt defining its behavior
  3. Select tools (summarize, web search, email, etc.)
  4. Choose a model or leave it on NICE Auto
  5. Click Create

Custom agents start at Common rarity and can evolve through usage milestones.

Spaceship Builder

Go to Bridge > Spaceships > New to build a custom spaceship:

  1. Describe your business or workflow
  2. NICE auto-generates a crew of agents tailored to your description
  3. Review and customize the crew
  4. Deploy

Sharing Blueprints

Share any blueprint via a unique 8-character link code. Shared links expire after 30 days.

To share: open a blueprint detail view and click Share. The recipient can import it into their own catalog.

Blueprint Markdown

Blueprints can be exported as markdown for version control, backup, or sharing outside the platform. The markdown format includes all fields in a human-readable structure.

Import a markdown blueprint via Bridge > Import.
