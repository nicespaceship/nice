Missions & Workflows

Missions are the unit of work in NICE. Every time you ask an agent to do something, it creates a mission. Workflows chain missions together.

Running a Mission

Open the Prompt Panel at the bottom of the screen (or press Cmd+K) and type your request. NICE does the rest:

  1. Routing — MissionRouter picks the best agent on your active spaceship for the task
  2. Model selection — the agent's preferred model is chosen (or NICE Auto picks one)
  3. Execution — the agent runs the task, optionally calling tools (web search, email, etc.)
  4. Result — output appears in the Missions tab with full markdown rendering

Mission Statuses

StatusMeaning
queuedMission created, waiting to start
runningAgent is actively working on the task
completedTask finished successfully
failedSomething went wrong (click to see error details)

Mission Actions

  • Run — execute a single mission
  • Run All — execute all queued missions on a spaceship
  • Retry — re-run a failed mission with the same parameters

Completed missions show a model badge indicating which AI model was used.

Streaming

Premium models (Claude, GPT-5) support streaming responses via Server-Sent Events. You see the agent's output in real time as it types. Free models (Gemini Flash) return the full response once complete.

Workflows

Workflows chain multiple missions into a sequence. Each step can use a different agent and pass its output to the next step.

Example workflow for a marketing campaign:

  1. Research Agent — analyze competitor campaigns
  2. Strategy Agent — draft a campaign plan based on research
  3. Content Agent — write copy following the plan
  4. Review Agent — proofread and refine

Scheduled Missions

Missions can run on a schedule — daily, weekly, or custom. Set up recurring tasks like:

  • Daily email digest at 9am
  • Weekly analytics report every Monday
  • Social media content generation every weekday

Schedules are managed in the Mission Scheduler, accessible from any spaceship detail view.

Token Usage

Each mission consumes tokens based on the model used. Free models (Gemini Flash) cost zero tokens. Premium models deduct from your balance. Check usage in Security > Wallet.
