Core Concepts

NICE organizes AI agents into a fleet hierarchy: Spaceships command crews of Agents, powered by Blueprints.

Spaceships

A Spaceship is your top-level orchestrator. It represents a complete workflow — a business, a department, or a project. When you deploy a Spaceship, it creates a crew of AI agents configured for that operation.

Spaceships have slots that hold agents. The number of slots depends on the ship's class:

Agents

Agents are the workers. Each agent has:

  • Role — analytics, content, marketing, sales, support, engineering, etc.
  • Tools — capabilities like summarize, web search, email, calendar, file generation
  • AI Model — the LLM powering the agent (assigned automatically or manually)
  • System Prompt — instructions that define the agent's behavior and personality

Agents execute missions — real tasks like drafting emails, analyzing data, scheduling posts, or generating reports.

Rarity

Every blueprint has a rarity level that determines when you can activate it:

All blueprints are visible to browse regardless of rank. Rarity only gates activation. Mythic blueprints can never be purchased — they must be earned through milestones.

XP & Ranks

You earn XP by using the platform. Every action counts:

  • Create an agent: 20 XP
  • Complete a mission: 15 XP
  • Chat with an agent: 5 XP
  • Create a workflow: 20 XP

There are 12 ranks from Ensign (0 XP) to Fleet Admiral (2.5M XP). Each rank unlocks higher ship classes, more slots, and rarer blueprints.

Missions

A mission is a task you give to your agents. Missions flow through this lifecycle:

  1. Queued — waiting to be picked up
  2. Running — an agent is working on it
  3. Completed — results are ready
  4. Failed — something went wrong (retryable)

NICE automatically routes each mission to the best agent and model on your crew. Results are rendered as formatted text with headers, lists, and code blocks.
