How NICE Auto Works

When you set a model to NICE Auto, the platform picks the optimal model for each task based on:

  • Task complexity — simple queries go to fast models, complex analysis goes to powerful ones
  • Available tokens — if your balance is low, NICE prefers free models
  • Historical performance — NICE learns which models perform best for your specific workflows

You can also manually assign a model to any agent in the Agent Builder.

Token System

Free models (Gemini Flash, Gemini Lite) never cost tokens. Premium models consume tokens per request.

Every new account gets 100K free tokens. After that, purchase token packages:

PackagePrice~Requests
500K tokens$4.99~250
5M tokens$19.99~2,500
25M tokens$49.99~12,500

Token usage is tracked per request. The nice-ai edge function checks your balance before making premium calls and returns a 402 if insufficient.

Enabling Models

Models are toggled on/off in Security > Integrations. Only enabled models are available for agent assignment and auto-routing.

Free tier users have Gemini Flash enabled by default. Pro subscribers can enable all 17 models.
