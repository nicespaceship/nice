Switching Themes

Click the theme button in the sidebar footer or go to Settings. Your choice persists across sessions.

Some themes change more than colors — The Office replaces sci-fi terminology with corporate language (Spaceships become "Departments", Agents become "Employees").

Theme Creator

Build your own theme at #/theme-editor. The editor gives you live controls for every CSS variable:

  • Colors — background, text, accent, borders, panels, glow effects
  • Typography — heading, body, and monospace font families
  • Shapes — border radius for cards, buttons, and panels

Changes preview instantly. Export your theme as JSON to share with others or import themes from the community.

--font-h, --font-b, --font-mHeading, body, monospace fonts
--radiusGlobal border radius

Theme-Aware Development

When building custom components or contributing to NICE:

  • Always use var(--accent) instead of hardcoded colors
  • Use var(--panel-bg) for card and panel backgrounds
  • Test your changes across at least 3 themes (NICE, The Office, Matrix)
  • Sticky elements need opaque backgrounds — never use rgba transparency on sticky headers
