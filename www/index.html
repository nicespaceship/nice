NICE (Neural Intelligence Command Engine) is an open-source AI agent platform. Deploy a crew of agents from 500+ blueprints. Connect them to 19+ services via MCP. They run real work for you: email, calendars, CRM, GitHub, Slack, and more. NICE provides the LLM, so you never deal with API keys.
A blueprint is a pre-built agent or spaceship template. Each one ships with a system prompt, role, and capabilities. Deploy it in one click. Browse 500+ blueprints across marketing, sales, engineering, and operations, or build your own.
An integration plugs your agents into your tools: Google Workspace, Microsoft 365, GitHub, Slack, HubSpot, Stripe, Notion, Linear, and more. NICE uses MCP (Model Context Protocol) servers to call APIs for you. Connect once via OAuth. Every agent on the spaceship inherits the access.
